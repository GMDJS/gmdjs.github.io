Миксин +grid()

Для того чтобы начать делать адаптивную верстку, вам необходимо создать контейнер, в котором вы сможете задавать элеменам сколько колонок они должны занимать внутри контейнера. Для этого в стилях добавьте миксин +grid() к любому селектору. Это может быть class, id, tag.

Этот миксин принимает один из двух аргументов +grid(fluid) или +grid(fixed). Агрумент fixed сделает вашу сетку фиксированной, когда размер контейнера для каждого медиа запроса фиксирован. А в случае передачи аргумента fluid контейнер будет растянут на всю ширину экрана браузера.

По умолчанию стоит значение fixed то есть вам достаточно применить миксин +grid() чтобы получить фиксированную сетку.

HTML

<div class="your-class">
  <div> ... </div>
  <div> ... </div>
  <div> ... </div>
</div>
Sass (sass syntax)

.your-class
  +grid()

Миксин +col()

Для каждого элемента родительского дива, заданным контейнером сетки, следует использовать миксин +col() c суффиксами или без. Этот миксин без суффикса указывает размер для елемента глобально

Чтобы задать элементу размер для конкретного разрешения добавьте суффиксы -xs -sm -md -lg -xl
Количестов колонок, которое займет элемент, задается аргументом в скобках. Например +col-md(5) - будет занимать 5 колонок на средних экранах.

Миксин +align()

Этот миксин служит для выравнивания элеменовт внутри контейнера и применяется к контейнеру. Его работа происходит за счет flexbox свойств justify-content и align-items.

Первым аргументом элементы позиционируются по горизонтали. Принимаются значения start, end, center, around, between.
Вторым аргументом позиционирование происходит по вертикали. Принимаются значения stretch, start, end, center, baseline.
По умолчанию установлены те же значения что и у свойств justify-content и align-items. То есть миксин +align() без аргументов будет равен +align(start, stretch)

Вы так же можете выравнивать элементы внутри контейнера только для нужного разрешения, это осуществляется путем добавления суффиксов -xs -sm -md -lg -xl. Без суффиксов миксин будет действовать для всех разрешений.

Миксин +offset()

Миксин +offset() задается для элементов контейнера. Он сдвигает элементы на нужное количество колонок в право или лево. Принимает два аргумента, первый аргумент определяет направление (right/left), а второй аргумент принимает число, на которое необходимо сдвинуть элемент. Например +offset(right, 5)

Он так же имеет суффиксы -xs -sm -md -lg -xl для всех медиазапросов.

Миксин +hidden()

Миксин +hidden() служит для скрытия элементов для нужных разрешений экрана с суффиксов -xs -sm -md -lg -xl. Просто укажите нужному элементу миксин +hidden() c нужнмы суффиксом и на этом разрешении экрана элемент будет скрыт. Миксин принимает однин аргумент со значениями:

only - элемент будет скрыт для текущего разрешения экрана, указанного суффиксом в миксине
up - элемент будет скрыт для текущего разрешения и всех больших
down - элемент будет скрыт для текущего разрешения и всех меньших

Миксин +media()

Ecли вам нужно применить какие-то свои стили для определенных размеров экрана, вы можете воспользоваться миксином +media() с нужным суффиксом -xs -sm -md -lg -xl . После него добавляйте любые свои стили, они применятся только для заданного размера экрана. Миксин принимает однин аргумент со значениями:

only - медиа запрос для текущего разрешения экрана, указанного суффиксом в миксине
up - медиа запрос для текущего разрешения и всех больших
down - медиа запрос для текущего разрешения и всех меньших

Настройка

Сетка имеет ряд настроек. Все настройкри производятся в файле settings.sass. Вы можете задать свои брейпоинты, ширину между колонками и их количество. В скором времени появится новые полезные настройки для кастомизации сетки.
