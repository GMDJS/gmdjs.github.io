Mixin +grid()

Любой элемент можно сделать контейнером, для этого добавьте миксин +grid() c суффиксами или без. Этот миксин без суффикса указывает размер для елемента глобально

Чтобы задать элементу размер для конкретного разрешения добавьте суффиксы -xs -sm -md -lg -xl
Сколько колонок будет занимать элемент определяет число в скобках. Например +col-md(5) - будет занимать 5 колонок на средних экранах.

<div class="grid">

  <div class="elem_2">2</div>
</div>

<div class="grid">
  <div class="elem">auto</div>
  <div class="elem">auto</div>
  <div class="elem">auto</div>
</div>
Sass

.grid
  +grid()
  .elem
    +col()
  .elem_3
    +col(3)
    +col-xs(12)
  .elem_7
    +col(7)
    +col-xs(12)
  .elem_2
    +col(2)
    +col-xs(12)

Mixin +col()

Для каждого элемента родительского дива, заданным контейнером сетки, следует использовать миксин +col() c суффиксами или без. Этот миксин без суффикса указывает размер для елемента глобально

Чтобы задать элементу размер для конкретного разрешения добавьте суффиксы -xs -sm -md -lg -xl
Количестов колонок, которое займет элемент, задается аргументом в скобках. Например +col-md(5) - будет занимать 5 колонок на средних экранах.
